JSU Feature Description
Tooltip Floating/Fixed Tips with GIF, Video, Code Highlight (JS, CSS, Java, Shell,...)
Validate API for both Item Validation and also Presentation of Validation Errors (in Red Items, Section, Popup)
LoadingDiv Loading Div to display during Long Operation. Many Options: Loading Gif, Title, Message, ElapsedSec...
Google Analytics API to build in few minutes a User Friendly List of Links to Google Analytics Pages
JSLog Log from JS Code with various LogLevel into an optional Window displayed adding an URL parameter.
SortTable Sort HTML Table by clicking column header
IE Popup Modal Blocking Popup only for IE. Blocking Code, based on IE ShowModalDialog
JQ Popup Modal Popup for whatever Browser. Not Blocking code with callback function
